# 图标风格切换说明

网站提供两种图标风格，可以根据需要切换。

## 🎨 两种风格对比

### 风格1：正常版（当前使用）
- **背景**：浅橘色 `bg-primary-light`
- **图标**：橙色 `text-primary`
- **效果**：柔和、清新
- **适用**：产品卡片、优势展示

```html
<div class="w-12 h-12 rounded-full flex items-center justify-center mb-4 bg-primary-light">
    <svg class="w-6 h-6 text-primary" fill="none" stroke="currentColor" viewBox="0 0 24 24">
        <!-- SVG 路径 -->
    </svg>
</div>
```

### 风格2：反显版
- **背景**：橙色 `bg-primary`
- **图标**：白色 `text-white`
- **效果**：醒目、强调
- **适用**：重要功能、CTA区域

```html
<div class="w-12 h-12 rounded-full flex items-center justify-center mb-4 bg-primary">
    <svg class="w-6 h-6 text-white" fill="none" stroke="currentColor" viewBox="0 0 24 24">
        <!-- SVG 路径 -->
    </svg>
</div>
```

---

## 🔄 如何切换风格

### 方法1：全局切换（推荐）

在 `index.html` 中搜索所有：
```html
bg-primary-light">
    <svg class="w-6 h-6 text-primary"
```

替换为：
```html
bg-primary">
    <svg class="w-6 h-6 text-white"
```

### 方法2：单个切换

找到想要修改的图标，修改两个地方：
1. 背景：`bg-primary-light` → `bg-primary`
2. 图标颜色：`text-primary` → `text-white`

## 🎯 快速替换代码

### 产品区域 - 改为反显版

搜索产品区域的图标，将：
```html
<!-- 正常版：浅橘色背景 + 橙色图标 -->
<div class="w-12 h-12 rounded-full flex items-center justify-center mb-4 bg-primary-light">
    <svg class="w-6 h-6 text-primary"
```

改为：
```html
<!-- 反显版：橙色背景 + 白色图标 -->
<div class="w-12 h-12 rounded-full flex items-center justify-center mb-4 bg-primary">
    <svg class="w-6 h-6 text-white"
```

### 关于我们区域 - 改为反显版

同样的方法，修改关于我们区域的3个图标。

## 🌈 颜色变量说明

所有颜色都使用 CSS 变量，修改主题色会自动应用到图标：

```css
:root {
    --primary-color: #FF6B00;      /* 橙色 */
    --primary-light: #FFF4E6;      /* 浅橘色 */
}
```

如果改成科技蓝：
```css
:root {
    --primary-color: #2563EB;      /* 蓝色 */
    --primary-light: #EFF6FF;      /* 浅蓝色 */
}
```

图标会自动变成蓝色系！

## 📝 注意事项

1. **保持一致性**：同一区域的图标建议使用同一风格
2. **对比度**：反显版对比度更高，更醒目
3. **主题色**：两种风格都会跟随主题色变化
4. **可访问性**：两种风格都符合 WCAG 对比度标准
