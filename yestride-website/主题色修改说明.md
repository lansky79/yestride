# 网站样式配置说明

## 📝 快速配置指南

所有颜色和字体大小都已统一为 CSS 变量，只需修改 `index.html` 中的 `<style>` 标签即可全局生效。

---

## 🎨 一、主题色配置

### 修改位置

打开 `index.html`，找到 `<style>` 标签中的这段代码：

```css
:root {
    /* ========== 主题色配置 ========== */
    --primary-color: #FF6B00;      /* 主题色 */
    --primary-dark: #E05E00;       /* 主题色深色 */
    --primary-light: #FFF4E6;      /* 主题色浅色背景 */
}
```

### 常用颜色方案

| 颜色 | primary-color | primary-dark | primary-light |
|------|---------------|--------------|---------------|
| 🟠 橙色（当前） | #FF6B00 | #E05E00 | #FFF4E6 |
| 🔵 科技蓝 | #2563EB | #1D4ED8 | #EFF6FF |
| 🟢 绿色 | #10B981 | #059669 | #F0FDF4 |
| 🟣 紫色 | #8B5CF6 | #7C3AED | #F5F3FF |

## 📏 二、字体大小配置

### 配置位置

同样在 `<style>` 标签中：

```css
:root {
    /* ========== 字体大小配置 ========== */
    /* Header 顶部导航栏 */
    --font-logo: 1.25rem;          /* Logo "Yestride" (默认 20px) */
    --font-company-name: 1rem;     /* "源踔科技" (默认 16px) */
    --font-nav-menu: 1rem;         /* "产品/案例/关于" (默认 16px) */
    --font-cta-button: 0.875rem;   /* "立即咨询" 按钮 (默认 14px) */
    
    /* 内容区域 */
    --font-hero-title: 3rem;       /* 主标题 (默认 48px) */
    --font-hero-subtitle: 1.125rem;/* 副标题 (默认 18px) */
    --font-section-title: 1.875rem;/* 章节标题 (默认 30px) */
    --font-card-title: 1.125rem;   /* 卡片标题 (默认 18px) */
    --font-body: 0.875rem;         /* 正文内容 (默认 14px) */
    
    /* 字重配置 */
    --weight-logo: 700;            /* Logo 字重 (粗体) */
    --weight-company-name: 600;    /* 公司名字重 (半粗) */
    --weight-nav-menu: 400;        /* 菜单字重 (正常) */
}
```

### 字体大小换算表

| rem | px | 说明 |
|-----|----|----|
| 0.75rem | 12px | 很小 |
| 0.875rem | 14px | 小 |
| 1rem | 16px | 正常 |
| 1.125rem | 18px | 稍大 |
| 1.25rem | 20px | 大 |
| 1.5rem | 24px | 较大 |
| 1.875rem | 30px | 很大 |
| 2rem | 32px | 超大 |
| 3rem | 48px | 标题 |

### 常用调整示例

#### 示例1：放大导航菜单
```css
--font-nav-menu: 1.125rem;  /* 从 16px 改为 18px */
```

#### 示例2：加粗公司名
```css
--weight-company-name: 700;  /* 从 600 改为 700 (更粗) */
```

#### 示例3：缩小 Logo
```css
--font-logo: 1rem;  /* 从 20px 改为 16px */
```

## 💡 快速调整技巧

### 想让整个网站字体都大一号？

在 `:root` 中统一调整：
```css
--font-logo: 1.5rem;           /* +0.25rem */
--font-company-name: 1.25rem;  /* +0.25rem */
--font-nav-menu: 1.125rem;     /* +0.125rem */
--font-hero-title: 3.5rem;     /* +0.5rem */
```

### 想让导航栏更紧凑？

```css
--font-logo: 1rem;             /* 缩小 Logo */
--font-company-name: 0.875rem; /* 缩小公司名 */
--font-nav-menu: 0.875rem;     /* 缩小菜单 */
```

### 想让标题更醒目？

```css
--font-hero-title: 4rem;       /* 放大主标题 */
--font-section-title: 2.25rem; /* 放大章节标题 */
```
