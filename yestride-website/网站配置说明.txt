===========================================
Yestride 网站配置说明
===========================================

这是一个完全静态的网站，所有内容都在 index.html 文件中。
你可以用任何文本编辑器（记事本、VS Code等）打开 index.html 进行修改。

===========================================
如何修改内容
===========================================

1. 用记事本或其他编辑器打开 index.html
2. 按 Ctrl+F 搜索要修改的文字
3. 直接修改
4. 保存文件
5. 双击 index.html 查看效果
6. 满意后上传到服务器

快速修改示例
===========================================

例如要修改公司电话：
1. 打开 index.html
2. 按 Ctrl+F 搜索 "400-xxx-xxxx"
3. 改成你的真实电话号码
4. 保存

例如要修改统计数字：
1. 打开 index.html
2. 搜索 "50+" 改成你想要的数字
3. 搜索 "成功案例" 改成你想要的标签
4. 保存

===========================================
注意事项
===========================================

1. 修改时不要删除HTML标签（<div>、<p>等）
2. 只修改标签之间的文字内容
3. 修改后先在本地双击查看效果
4. 确认无误后再上传到服务器
5. 建议修改前先备份原文件
