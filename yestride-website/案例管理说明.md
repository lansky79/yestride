## 📝 如何编辑案例

### 修改案例标题和描述

在 `index.html` 中找到对应的案例：

```html
<!-- Case 1: 智慧矿山 -->
<div class="case-card group">
    <div class="relative overflow-hidden rounded-lg mb-4">
        <img src="images/cases/case-1.jpg" alt="智慧矿山综合管控平台" 
             class="w-full aspect-video object-cover transition-transform duration-500 group-hover:scale-110">
        <!-- 占位符（图片加载失败时显示） -->
        <div class="bg-gradient-to-br from-gray-300 to-gray-400 aspect-video rounded-lg hidden items-center justify-center">
            <span class="text-gray-500 text-lg">智慧矿山综合管控平台</span>
        </div>
        <!-- 悬停遮罩 -->
        <div class="absolute inset-0 bg-gradient-to-t from-black/60 via-transparent to-transparent opacity-0 group-hover:opacity-100 transition-opacity duration-300"></div>
    </div>
    <!-- 标题 -->
    <h3 class="font-bold text-xl mb-2 group-hover:text-primary transition-colors">智慧矿山综合管控平台</h3>
    <!-- 描述 -->
    <p class="text-gray-600 text-sm leading-relaxed">集成矿山生产、安全、设备等多维度数据，实现智能化综合管控</p>
</div>
```

修改三个地方：
1. `alt` 属性：图片描述
2. `<h3>` 标题
3. `<p>` 描述文字

## ➕ 添加新案例

### 步骤1：准备图片
将图片放到 `images/cases/` 文件夹，命名为 `case-11.jpg`

### 步骤2：复制案例代码
在 `index.html` 中复制一个案例块，修改内容：

```html
<!-- Case 11: 新案例 -->

    <div class="relative overflow-hidden rounded-lg mb-4">
        <img src="images/cases/case-11.jpg" alt="新案例标题" 
             class="w-full aspect-video object-cover transition-transform duration-500 group-hover:scale-110" 
             onerror="this.style.display='none'; this.nextElementSibling.style.display='flex'">
        <div class="bg-gradient-to-br from-gray-300 to-gray-400 aspect-video rounded-lg hidden items-center justify-center">
            <span class="text-gray-500 text-lg">新案例标题</span>
        </div>
        <div class="absolute inset-0 bg-gradient-to-t from-black/60 via-transparent to-transparent opacity-0 group-hover:opacity-100 transition-opacity duration-300"></div>
    </div>
    <h3 class="font-bold text-xl mb-2 group-hover:text-primary transition-colors">新案例标题</h3>
    <p class="text-gray-600 text-sm leading-relaxed">新案例的描述文字...</p>
</div>
```

### 步骤3：测试
刷新页面，检查新案例是否正常显示和翻页。

## 🎯 轮播设置

### 当前配置
- **每页显示**：3个案例
- **自动隐藏箭头**：案例 ≤ 3 个时
- **指示器圆点**：显示当前页码

### 修改每页显示数量

在 `js/main.js` 中修改：

```javascript
const cardsPerPage = 3; // 改为 4 或其他数字
```

可选值：
- `2` = 每页2个（2列）
- `3` = 每页3个（3列，当前）
- `4` = 每页4个（4列）

## 🎨 自定义样式

### 修改图片高度

在 `css/style.css` 中：

```css
.aspect-video {
    aspect-ratio: 16 / 9;  /* 16:9 宽屏 */
}

/* 改为 4:3 */
.aspect-video {
    aspect-ratio: 4 / 3;
}
```

### 修改悬停效果

```css
.case-card:hover {
    transform: translateY(-8px);  /* 上浮距离 */
    box-shadow: 0 20px 40px rgba(0,0,0,0.12);  /* 阴影 */
}
```

### 修改卡片圆角

```css
.case-card {
    border-radius: 12px;  /* 改为 8px 或 16px */
}
```

## ❓ 常见问题

### Q: 图片不显示怎么办？
A: 检查：
1. 图片路径是否正确：`images/cases/case-X.jpg`
2. 图片文件是否存在
3. 图片格式是否支持（JPG/PNG/WebP）
4. 如果图片不存在，会显示灰色占位符

### Q: 轮播不工作？
A: 检查：
1. `js/main.js` 是否正确加载
2. 浏览器控制台是否有错误
3. 案例数量是否 > 3 个（≤3个会隐藏箭头）

### Q: 如何改回4列布局？
A: 在 `index.html` 中找到：
```html
<div id="cases-container" class="grid grid-cols-1 md:grid-cols-2 lg:grid-cols-3 gap-6">
```
改为：
```html
<div id="cases-container" class="grid grid-cols-1 md:grid-cols-2 lg:grid-cols-4 gap-6">
```
同时在 `js/main.js` 中改为 `const cardsPerPage = 4;`

### Q: 如何去掉悬停效果？
A: 在 `css/style.css` 中删除或注释掉：
```css
.case-card:hover {
    transform: translateY(-8px);
    box-shadow: 0 20px 40px rgba(0,0,0,0.12);
}
```
