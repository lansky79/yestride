# Yestride 网站编辑器使用说明

## 🚀 模块配置版 - 完全控制网站显示内容！

这是一个功能强大的可视化编辑器，支持模块开关控制和案例管理，让你可以自由定制企业网站的每个部分。

## ✨ 三个版本

### 1. 基础版 (editor.html)
- 简单的表单编辑
- 适合快速修改基本信息

### 2. 强大版 (editor-v2.html)
- 支持所有内容编辑
- 可以添加/删除各种模块

### 3. 模块配置版 (editor-v3.html) ⭐ **推荐使用**
- **模块开关控制**：选择显示/隐藏各个模块
- **首屏细节控制**：控制NEW标签、按钮、客户头像、评分等
- **案例管理系统**：添加案例、上传图片、管理标签和亮点
- **完全自定义**：保留需要的内容，隐藏不需要的部分

## 🎯 新功能亮点

### 模块开关控制
可以控制以下模块的显示/隐藏：
- ✅ 首屏区域（公司介绍和Slogan）

- ✅ 业务模块（核心业务介绍）
- ✅ 案例展示（成功案例）

- ✅ 合作流程（合作步骤）

### 首屏细节控制
- 显示/隐藏 NEW 标签
- 显示/隐藏"立即咨询"和"了解更多"按钮
- 显示/隐藏客户头像圆圈
- 显示/隐藏"已服务XX客户"文字

每个案例可以配置：

- 分类（政务/金融/营销等）
- 案例描述
- 图片路径（支持上传）
- 标签（多个标签）
- 项目亮点（多个亮点）

## 📦 安装步骤

### 1. 安装Node.js
如果还没有安装Node.js，请先安装：
- 访问 https://nodejs.org/
- 下载并安装LTS版本

### 2. 安装依赖
在编辑器目录打开命令行，运行：
```bash
npm install
```

### 3. 启动编辑器
```bash
npm start
```

看到提示后，在浏览器中打开：
- 基础版：http://localhost:3000/editor.html
- 强大版：http://localhost:3000/editor-v2.html
- 模块配置版：http://localhost:3000/editor-v3.html ⭐

## 🎨 使用步骤

### 1. 启动编辑器
运行 `npm start`，然后打开浏览器访问模块配置版

### 2. 配置模块显示
在"模块开关"标签页：
- 开启需要显示的模块
- 关闭不需要的模块
- 配置首屏区域的细节显示

### 3. 编辑内容
根据需要编辑各个标签页的内容：
- **基本信息**：公司名称、Slogan、描述
- **首屏区域**：标题、副标题、NEW标签
- **技术栈**：技术名称和颜色（可增删）
- **业务模块**：业务标题、描述、特点（可增删）
- **案例展示**：添加案例、上传图片、配置标签和亮点 ⭐
- **优势展示**：优势标题和描述（可增删）
- **合作流程**：流程步骤（可增删）
- **联系方式**：电话、邮箱、地址、CTA文案

### 4. 添加案例
在"案例展示"标签页：
1. 点击"+ 添加案例"
2. 填写案例标题、分类、描述
3. 输入图片路径（如：images/cases/case-1.jpg）
4. 添加标签（用逗号分隔）
5. 添加亮点（每行一个）

### 5. 保存和生成
1. 点击"💾 保存配置"保存到config.json
2. 点击"🚀 生成网站"生成HTML文件
3. 点击"👁️ 预览网站"查看效果

### 6. 上传到服务器
将 `output/index.html` 和相关图片文件上传到服务器


## 💡 使用建议

### 根据需求定制
1. **保留核心内容**：公司介绍、Slogan、技术栈
2. **隐藏不需要的**：关闭"立即咨询"按钮、客户头像、评分等
3. **添加案例展示**：展示成功项目，增加可信度
4. **简化首屏**：只保留必要信息，避免信息过载

### 案例图片管理
1. 将案例图片放在 `output/images/cases/` 目录
2. 图片命名建议：case-1.jpg, case-2.jpg
3. 推荐尺寸：1200x800 或 16:9 比例
4. 图片格式：JPG 或 PNG

### 配置示例

#### 简洁版配置
- 开启：首屏、技术栈、业务模块、案例展示
- 关闭：客户头像、评分、数据统计
- 首屏只显示：标题、描述、NEW标签

#### 完整版配置
- 开启所有模块
- 首屏显示所有元素
- 添加多个案例展示

## 🔧 常见问题

### Q: 如何隐藏"立即咨询"按钮？
A: 在"模块开关"标签页，取消勾选"显示'立即咨询'按钮"

### Q: 如何添加案例图片？
A: 在"案例展示"标签页，输入图片路径（如：images/cases/case-1.jpg），然后将图片文件放到output/images/cases/目录

### Q: 案例的标签和亮点有什么区别？
A: 标签是简短的关键词（如"AI大模型"），亮点是具体的成果描述（如"服务事项在线办理率提升至95%"）

### Q: 可以只显示部分模块吗？
A: 可以！在"模块开关"标签页关闭不需要的模块即可

### Q: 生成的网站会包含隐藏的模块吗？
A: 不会！关闭的模块不会出现在生成的HTML中

### Q: 如何恢复默认配置？
A: 点击"🔄 重新加载"按钮，会从config.json重新加载配置

## ⚠️ 注意事项

1. **模块依赖**：某些模块可能依赖其他模块，建议保留核心模块
2. **图片路径**：确保图片路径正确，相对于output目录
3. **配置备份**：修改前建议备份config.json文件
4. **浏览器兼容**：推荐使用Chrome、Edge、Firefox等现代浏览器
