# Yestride 网站编辑器 - 快速上手指南

## 🎯 你的需求

根据你的要求，我创建了一个**模块配置版编辑器**，可以：

✅ **保留原有内容**：公司介绍和Slogan都保留  
✅ **隐藏不需要的**：可以关闭"立即咨询"、"了解更多"按钮  
✅ **隐藏客户展示**：可以关闭客户头像、评分、"已服务XX客户"  
✅ **保留技术栈**：AI大模型、云原生架构等保留  
✅ **添加案例**：可以自己添加案例，包括图片  

## 🚀 快速开始

### 1. 启动编辑器
```bash
npm start
```

### 2. 打开浏览器
访问：http://localhost:3000/editor-v3.html

### 3. 配置模块显示

在**"模块开关"**标签页：

#### 首屏区域细节控制
- ✅ 勾选"显示NEW标签" - 保留
- ❌ 取消"显示'立即咨询'按钮" - 不要按钮
- ❌ 取消"显示客户头像" - 不要头像
- ❌ 取消"显示'已服务XX客户'" - 不要客户统计
- ❌ 取消"显示评分和评价" - 不要评价

#### 模块开关
- ✅ 首屏区域 - 保留（公司介绍和Slogan）
- ✅ 技术栈展示 - 保留（AI大模型等）
- ✅ 业务模块 - 保留
- ✅ 案例展示 - 保留（可以添加案例）
- ✅ 优势展示 - 保留
- ❌ 数据统计 - 关闭（不要"已服务多少客户"）
- ✅ 合作流程 - 保留
- ✅ CTA区域 - 保留

### 4. 添加案例

在**"案例展示"**标签页：

1. 点击"+ 添加案例"
2. 填写信息：
   - **案例标题**：如"某市政府智慧政务平台"
   - **分类**：如"政务"、"金融"、"营销"
   - **案例描述**：简短描述项目内容
   - **图片路径**：如 `images/cases/case-1.jpg`
   - **标签**：如"AI大模型, 云原生, 政务服务"（用逗号分隔）
   - **亮点**：每行一个，如：
     ```
     服务事项在线办理率提升至95%
     群众满意度达98%
     办理时长缩短60%
     ```

3. 准备图片：
   - 将案例图片放到 `output/images/cases/` 目录
   - 图片命名：case-1.jpg, case-2.jpg 等
   - 推荐尺寸：1200x800 或 16:9 比例

### 5. 保存和生成

1. 点击"💾 保存配置"
2. 点击"🚀 生成网站"
3. 点击"👁️ 预览网站"查看效果
