# 案例图片目录

将你的案例图片放在这个目录下。

## 图片要求

- **格式**：JPG 或 PNG
- **尺寸**：推荐 1200x800 像素（16:9 比例）
- **命名**：建议使用有意义的名称，如：
  - `gov-platform.jpg` - 政府平台项目
  - `bank-ai.jpg` - 银行AI项目
  - `digital-human.jpg` - 数字人项目

## 使用方法

1. 将图片文件复制到这个目录
2. 在编辑器的"案例展示"标签页中
3. 在"图片路径"字段填写：`images/cases/你的图片名.jpg`

## 示例

如果你的图片文件是 `gov-platform.jpg`，那么在编辑器中填写：
```
images/cases/gov-platform.jpg
```

## 注意事项

- 图片文件名不要包含中文或特殊字符
- 图片大小建议控制在 500KB 以内
- 确保图片清晰，适合展示
