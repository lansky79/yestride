# Yestride 网站编辑器 - 功能说明

## 📌 核心功能

### 1. 模块开关控制 ⭐
可以自由控制网站显示哪些模块，不需要的模块可以直接关闭。

**支持的模块：**
- 首屏区域（Hero）
- 技术栈展示（Tech Stack）
- 业务模块（Services）
- 案例展示（Cases）⭐ 新增
- 优势展示（Advantages）
- 数据统计（Stats）
- 合作流程（Process）
- CTA区域（Call to Action）

### 2. 首屏细节控制 ⭐
可以精确控制首屏区域显示哪些元素：

- **NEW标签**：显示/隐藏"AI驱动的解决方案"标签
- **CTA按钮**：显示/隐藏"立即咨询"和"了解更多"按钮
- **客户头像**：显示/隐藏客户头像圆圈
- **客户统计**：显示/隐藏"已服务 50+ 政企客户"
- **评分评价**：显示/隐藏5星评分和评价数量

### 3. 案例管理系统 ⭐ 新增
完整的案例管理功能，支持：

**案例信息：**
- 案例标题
- 分类标签（政务/金融/营销等）
- 详细描述
- 案例图片（支持上传）

**案例标签：**
- 多个技术标签
- 用逗号分隔
- 自动显示为标签样式

**项目亮点：**
- 多个亮点描述
- 每行一个亮点
- 自动显示为列表

### 4. 内容编辑
所有文字内容都可以编辑：

- 公司信息（名称、Slogan、描述）
- 首屏内容（标题、副标题、标签）
- 技术栈（名称、颜色）
- 业务模块（标题、描述、特点）
- 优势展示（标题、描述）
- 合作流程（步骤、标题、描述）
- 联系方式（电话、邮箱、地址）
- CTA文案（标题、副标题、按钮文字）

### 5. 动态管理
支持添加、删除、修改各种模块：

- ➕ 添加新技术
- ➕ 添加新业务
- ➕ 添加新案例 ⭐
- ➕ 添加新优势
- ➕ 添加新流程步骤
- ❌ 删除任何模块

## 🎯 使用场景

### 场景1：简洁版网站
**需求**：只展示核心信息，不要太多营销元素

**配置**：
- ✅ 开启：首屏、技术栈、业务模块、案例展示
- ❌ 关闭：客户头像、评分、数据统计、CTA按钮
- 结果：简洁专业的企业网站

### 场景2：营销版网站
**需求**：展示所有内容，增加可信度

**配置**：
- ✅ 开启所有模块
- ✅ 显示所有首屏元素
- ✅ 添加多个案例展示
- 结果：内容丰富的营销型网站

### 场景3：案例展示版
**需求**：重点展示成功案例

**配置**：
- ✅ 开启：首屏、技术栈、案例展示、CTA
- ❌ 关闭：业务模块、优势展示、数据统计
- ✅ 添加多个详细案例
- 结果：以案例为核心的展示网站

## 🔧 技术特点

### 1. 模块化架构
- 每个模块独立控制
- 关闭的模块不会生成HTML代码
- 减少页面体积，提升加载速度

### 2. 配置驱动
- 所有配置保存在JSON文件
- 易于备份和恢复
- 支持版本控制

### 3. 实时预览
- 修改后立即保存
- 一键生成网站
- 浏览器预览效果

### 4. 图片管理
- 支持本地图片
- 自动处理图片路径
- 图片加载失败自动隐藏

## 📝 配置文件结构

```json
{
  "modules": {
    "hero": {
      "enabled": true,
      "showBadge": true,
      "showCTAButtons": false,
      "showSocialProof": false,
      "showClientStats": false,
      "showRatings": false
    },
    "techStack": { "enabled": true },
    "services": { "enabled": true },
    "cases": { "enabled": true },
    "advantages": { "enabled": true },
    "stats": { "enabled": false },
    "process": { "enabled": true },
    "cta": { "enabled": true }
  },
  "cases": [
    {
      "title": "案例标题",
      "category": "分类",
      "description": "描述",
      "image": "images/cases/case-1.jpg",
      "tags": ["标签1", "标签2"],
      "highlights": ["亮点1", "亮点2"]
    }
  ]
}
```

## 🎨 界面说明

### 标签页
1. **模块开关** - 控制显示哪些模块
2. **基本信息** - 公司名称、Slogan
3. **首屏区域** - 标题、副标题
4. **技术栈** - 技术名称和颜色
5. **业务模块** - 业务介绍
6. **案例展示** - 成功案例 ⭐
7. **优势展示** - 公司优势
8. **合作流程** - 合作步骤
9. **联系方式** - 联系信息和CTA

### 按钮功能
- **💾 保存配置** - 保存到config.json
- **🚀 生成网站** - 生成HTML文件
- **🔄 重新加载** - 从config.json重新加载
- **👁️ 预览网站** - 在新标签页预览

### 4. 性能优化
- 关闭不需要的模块
- 优化图片大小
- 减少不必要的内容

## 🚀 快速上手

1. **启动编辑器**
   ```bash
   npm start
   ```

2. **打开浏览器**
   访问：http://localhost:3000/editor-v3.html

3. **配置模块**
   在"模块开关"标签页配置显示内容

4. **添加案例**
   在"案例展示"标签页添加案例

5. **保存生成**
   点击"保存配置"和"生成网站"

6. **预览效果**
   点击"预览网站"查看效果
